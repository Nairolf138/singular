Guide de lecture rapide (humain)
  1. Cockpit : regardez d’abord le statut global, puis la prochaine action recommandée.
  2. Alertes et risques : vérifiez les zones en orange/rouge pour prioriser.
  3. Timeline : cliquez un événement pour comprendre le “pourquoi” d’une mutation.
  4. Vies : comparez les organismes pour décider quoi conserver, corriger ou archiver.
